## Develop Environment

### Flutter SDK: v1.9.1

### Android Studio: 3.5.1

#### Plugins:

* Flutter: MUST
* Dart: MUST
* Flutter i18n: recommended. It makes to easy Flutter localization.
* Kotlin: recommended
