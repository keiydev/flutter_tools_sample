## 開発環境

### Flutter SDK: v1.9.1

### Android Studio: 3.5.1

#### 使用Plugin: 

* Flutter: 必須
* Dart: 必須
* Flutter i18n: おすすめ。Flutterの多言語対応を楽にしてくれる。
* Kotlin: おすすめ。Android側のjavaからkotlinへの移行に。
